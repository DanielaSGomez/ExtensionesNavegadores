# 📝 Extensiones de Navegador

Este repositorio contiene mis extensiones de navegador desarrolladas en **JavaScript / HTML / CSS**, con estética pastel/kawaii y funcionalidades útiles para el día a día.

---

## 📌 Extensión: Notas Rápidas

### Descripción
Una extensión simple y kawaii para tomar notas rápidas directamente desde el navegador.  
Permite escribir, guardar, copiar y eliminar notas con un diseño pastel amigable.

### ✨ Características
- Guardar notas en el almacenamiento local del navegador.
- Visualizar todas las notas en un popup más amplio y cómodo.
- Botón ❌ para eliminar cada nota.
- Texto seleccionable para copiar fácilmente.
- Estilo pastel/kawaii con ícono personalizado.

### 📂 Estructura del proyecto
Notas/
├── manifest.json
├── popup.html
├── popup.js
├── style.css
└── icon.png


### ⚙️ Instalación en Chrome
1. Clonar este repositorio o descargar la carpeta de la extensión.
2. Abrir `chrome://extensions/`.
3. Activar **Modo desarrollador**.
4. Clic en **Cargar descomprimida** y seleccionar la carpeta `Notas`.
5. ¡Listo! El ícono aparecerá en la barra del navegador.
